Financial Management Information System (FMIS) is a web application for the Department of Energy (DOE). The images show the sample list, registration, and view of the system. It has a Report Generation Module for the summary and printing of reports.

We used Grails 2.4 framework in building the application. Materialized CSS framework for the design and PostgreSQL for the database management. We alse used Jasper Reports for generating PDF and Excel file. In terms of wireframing, we used Axure RP 8.

Online System for Energy Collaboration and Cooperation (OSECC) is the Phase 3 of the whole web application project. It is the integration of 3 systems namely, Online Service Management System (OSMS), Critical Energy Installation Information System (CEIIS), and Legal Services Information System (LSIS). It has an admin and client side. The images show the sample list, registration, and dashboard of both admin and client side.

We used the version of Phase 1 and Phase 2 in building the system to easily integrate the whole project. Grails 2.3.8 framework, Bootstrap 3 for the CSS framework. PostgreSQL for the database management. And also Jasper Reports for generating PDF and Excel file. For wireframing and designing, we used Axure RP 8.
